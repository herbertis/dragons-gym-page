# 🐉 Dragons Gym - Website Redesign & Lead Generation

[Preview do Projeto](https://herbertis.github.io/dragons-gym-page/)

---

## 💻 Sobre o Projeto

Este projeto consiste na **repaginação completa (redesign) da Landing Page da Dragons Gym**.

O objetivo principal foi modernizar a presença digital da academia, substituindo um layout antigo por uma interface **premium, imersiva e altamente focada em conversão (captação de novos alunos)**.

Através de um design **Dark Mode com detalhes em neon vermelho**, o site agora transmite a força e a energia da marca, enquanto a **arquitetura da informação guia o usuário diretamente para a escolha de planos e o preenchimento do formulário de matrícula**.

---

## 🚀 Principais Funcionalidades (Features)

* **Design Imersivo & Efeito Neon:**  
Uso estratégico de sombreamentos (`box-shadow`) e transições para criar um efeito neon interativo nos cards de planos e serviços.

* **Scroll Reveal Animations:**  
Implementação da `Intersection Observer API` em Vanilla JavaScript para criar animações de entrada fluidas (`fade-in`, `fade-left`, `fade-right`) conforme o usuário rola a página, aumentando o tempo de retenção.

* **Seleção Dinâmica de Planos:**  
Lógica em JavaScript para garantir que o usuário interaja com as opções de planos e serviços, destacando a escolha ativa e facilitando a decisão de compra.

* **Captação de Leads:**  
Seção dedicada com formulário otimizado para coletar dados de potenciais novos clientes diretamente no sistema da academia.

* **100% Responsivo:**  
Layout adaptável para qualquer tamanho de tela (**Mobile, Tablet e Desktop**), garantindo a melhor experiência (**UX**) em todos os dispositivos.

* **Botão Flutuante do WhatsApp:**  
Atalho rápido e animado (efeito `pulse`) para atendimento direto com a equipe de vendas.

---

## 🛠️ Tecnologias Utilizadas

Neste projeto, optei por uma abordagem **limpa e de alta performance**, sem dependência de frameworks pesados para o front-end:

* **HTML5:**  
Semântica e estruturação do conteúdo.

* **CSS3:**  
Flexbox, CSS Grid, animações (`@keyframes`), variáveis e design responsivo (`Media Queries`).

* **JavaScript (Vanilla):**  
Manipulação de DOM e uso da `Intersection Observer API` para performance de rolagem.

Como profissional que une **Design e Desenvolvimento Front-end**, o foco desta repaginação foi além do código estético.

O novo layout resolve os seguintes problemas de negócio:

1. **Aumento da Percepção de Valor:**  
O visual moderno justifica e valoriza os preços dos planos da academia.

2. **Melhoria na Taxa de Conversão:**
